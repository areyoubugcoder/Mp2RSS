# 服务介绍

**Mp2RSS 是一款把微信公众号转成标准 Feed 的订阅服务**，面向希望在任意阅读器里稳定、及时收到公众号更新的信息消费者、知识管理玩家以及行业观察者。

用户通过 GitHub 或 Google 登录后，即可搜索公众号或贴入任意一篇公众号文章链接把对应账号加入订阅。Mp2RSS 会持续收录被订阅公众号的最新文章，并以 RSS / Atom / JSON Feed / OPML 等主流格式对外分发，让你在 Reeder、NetNewsWire、FreshRSS、Miniflux 等任何阅读器里同步阅读。

## 核心流程

1. 用 GitHub / Google 账号登录 Mp2RSS；
2. 通过搜索或粘贴文章链接订阅公众号；
3. 复制 Feed 链接（单个公众号或聚合合集）贴入任意阅读器，即可持续收到更新。

## 服务特点

- **无需微信账号**：Mp2RSS 已对接稳定的数据源，你不需要登录微信或扫码授权；
- **主动聚合**：订阅后系统持续收录新文章，阅读器按自己的节奏拉取即可，无需人工搬运；
- **多格式并存**：RSS 2.0 / Atom 1.0 / JSON Feed 1.1 / OPML 2.0 任选，便于跨工具互通；
- **单号 + 合集**：既支持按公众号生成独立 Feed，也支持把全部订阅合并为一个聚合 Feed；
- **可复位的 Feed 密钥**：链接中的密钥随时一键重置，旧链接立即失效；
- **API 友好**：订阅管理、文章拉取都能通过标准 HTTP 调用完成，方便接入自动化工作流。
