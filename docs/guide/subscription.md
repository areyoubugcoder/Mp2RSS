# 订阅管理

## 添加订阅

两种方式：

- **按名称搜索**：输入公众号关键词，从候选结果中直接订阅；
- **文章链接反查**：贴入任意一篇公众号文章 URL，系统会反查其来源公众号并登记订阅。

## 订阅配额

- 单账户订阅上限 **400 个公众号**；

## 取消订阅

- 支持**单个取消**与**批量取消**；
- 取消后 Feed 会立即停止该公众号的后续更新，聚合 Feed 中也不再展示该公众号的新文章；

## 相关接口

- 通过 [API](/api/) 可以脚本化完成订阅 / 取消 / 列表查询 / 拉取文章；
- 通过 Feed 公开层链接可直接被阅读器订阅，详见 [快速开始](/guide/quick-start)。
