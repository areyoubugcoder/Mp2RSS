# 快速开始

四步把一个公众号订阅到你常用的阅读器里。

## 1. 登录账号

打开 Mp2RSS 控制台，选择 **使用 GitHub 登录** 或 **使用 Google 登录** 完成授权。

- 同一邮箱只会对应一个账户，先后用不同 OAuth 来源登录也会落到同一账户；
- 若 OAuth 回调未返回可用邮箱（例如 GitHub 邮箱设为私有、Google 账号未验证邮箱），登录会被拒绝。请先在 OAuth 提供方处公开 / 验证邮箱后重试。

登录成功后：

- 系统会为账户生成一个初始的 **Feed 密钥**；
- 自动开通 **免费试用**，权益与正式会员完全相同；
- 在"账户设置"页可查看 Feed 链接、剩余试用天数、订阅用量等信息。

## 2. 订阅公众号

进入"订阅管理"页：

- **粘贴文章链接**：贴入任意一篇公众号文章 URL（`https://mp.weixin.qq.com/s?...`），系统会自动登记订阅。

规则提醒：

- 单账户订阅上限 **400 个公众号**；

## 3. 复制 Feed 链接

Mp2RSS 为每个账户生成一组专属链接，链接中携带 Feed 密钥即可被任意阅读器公开访问，**无需登录**。

在"账户设置"页点击链接旁的复制按钮即可获取完整地址。

## 4. 导入阅读器

把上一步复制的链接粘贴到你的阅读器：

- **移动端 / 桌面阅读器**（Reeder、NetNewsWire、Fluent Reader 等）：新增订阅源，粘贴 RSS 或 Atom 链接；
- **自建 RSS 服务**（FreshRSS、Miniflux、Tiny Tiny RSS 等）：在"添加订阅"中粘贴链接，或导入 OPML 一次性迁移整份订阅清单；
- **JSON Feed 消费方**：直接使用 JSON Feed 1.1 链接，便于程序化处理。
