# 服务条款

> 本条款摘要说明 Mp2RSS 当前版本的使用规则与边界。最终版权与法律文本以控制台内正式发布的服务协议为准。

## 1. 服务内容

Mp2RSS 是一个把微信公众号内容转成标准 Feed（RSS / Atom / JSON Feed / OPML）的订阅服务。用户可在登录后通过界面或 Open API 管理订阅，并在任意阅读器中使用我们提供的 Feed 链接接收更新。

## 2. 账户

- 账户以邮箱作为唯一标识。同一邮箱在不同 OAuth 来源（GitHub / Google）登录都会落到同一账户；
- 若 OAuth 回调未返回可用邮箱，登录会被拒绝；
- 请自行妥善保管账户登录凭据，由于账户凭据泄漏引发的数据或权益损失，平台不承担责任；
- 管理员在发现严重滥用行为时，有权封禁账户，被封禁账户的登录、订阅管理及 Feed 链接即时失效。

## 3. Feed 密钥

- Feed 链接中携带的密钥等同于访问凭据，请勿公开分享；
- 用户可在控制台随时重置密钥，重置后旧链接**立即失效**；
- 若发现密钥泄漏，请立刻重置。

## 4. 使用规范

使用 Mp2RSS 服务时，请遵守以下规范：

- 不利用服务从事违法违规、侵犯他人知识产权或干扰他人合法权益的行为；
- 不通过异常高频请求、伪造身份等方式规避系统对订阅上限等规则的保护；
- 不用于非法爬取、二次售卖受版权保护的内容；
- Feed 链接的转发、分享行为由用户自行负责，因此引发的账户滥用由账户所有人承担。

## 5. 数据来源与时效

- Mp2RSS 不直接抓取公众号，内容由外部数据源提供；
- 从公众号发文到 Feed 可见存在秒级到分钟级的延迟；
- 平台尽力保证文章落库的完整性（markdown 正文完整后才入库），但不对个别文章的时效、样式、图文完整性提供 SLA 承诺；
- 当数据源发生临时不可用时，相关搜索 / 订阅 / 文章拉取请求会以 5xx 状态码反馈给调用方。

## 6. 付费、试用与到期

- 所有新注册账户自动获得 3 天免费试用期；
- 付费由月付（15 元 / 月）与年付（150 元 / 年）两档组成；
- 充值通过线下转账完成，到账核对后为你开通对应订阅计划；
- 会员到期（含试用到期）后进入只读观察期，Feed 中会注入"会员已到期"提示条目；
- 如对账单或开通状态有疑问，可通过控制台凭证上传入口提交凭证。

## 7. 内容版权与责任

- 所有公众号文章版权归原作者所有；Mp2RSS 仅作为订阅分发通道；
- 请用户在遵守相关平台规则与法律法规的前提下使用相关内容；
- 因转发、二次加工、商用等行为引发的版权纠纷由使用者自行承担。

## 8. 服务变更

- 为优化产品体验或响应合规要求，平台可能调整功能、配额、价格等参数；重大调整会在控制台公告；
- 为保障系统稳定，平台可能进行计划内或临时的维护；维护期间部分服务可能短暂不可用。

## 9. 免责

在适用法律允许的最大范围内，Mp2RSS 不对因使用本服务产生的间接损失承担责任，包括但不限于商业机会损失、数据延迟造成的决策影响等。
