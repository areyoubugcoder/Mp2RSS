# 会员与计费

## 免费试用

- 注册成功即自动获得 **3 天免费试用期**；
- 试用期内权益与正式会员**完全一致**：可订阅、生成全部格式 Feed、调用 Open API；
- 站内会在试用到期前显著提示续费入口。

## 套餐结构

Mp2RSS 仅提供一个套餐，覆盖全部功能权益，按计费周期分为两档：

- **月付**
- **年付**

具体价格以控制台实际展示为准。试用期与付费期权益完全相同，不存在"高级版"与"基础版"的差异。

## 续期规则

- **在有效期内续费**：新时长自动叠加在当前到期时间之上，不会浪费剩余时长；
- **已到期后续费**：从续费当日起算，不回补到期期间的空档；
- 月付与年付可混合叠加（例如当前为年付有效期内，续订一个月付会在年付到期日基础上再追加一个月）。

## 充值流程

1. 在控制台选择 **月付** 或 **年付**；
2. 页面展示收款说明与转账二维码，二维码区域会同时显示你的用户标识，请在转账备注里填写该标识以便核对；
3. 线下完成付款后，点击"**完成充值**"；
4. 我们核对入账后为你开通 / 续订对应的订阅计划，权益即时生效。

### 忘记点击"完成充值"或未正常续费？

控制台提供**凭证上传**入口作为兜底：

- 上传充值截图 + 用户标识，我们会据此核对并为你开通；

## 账户信息一览

在"账户设置"页可随时查看：

- Feed 链接（支持一键复制）；
- 重置 Feed 密钥（需二次确认）；
- 订阅用量 / 试用剩余天数 / 会员有效期；
- 近期的付费记录。

## 关于 Feed 密钥重置

重置后旧链接**立即失效**，所有外部阅读器里配置的旧链接都会报错。请在重置后及时用新链接重新在阅读器中订阅。
